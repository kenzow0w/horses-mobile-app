# StableTracker (минимальное приложение для конюшни)

Небольшое Android‑приложение на Kotlin (Jetpack Compose + Room) для учёта лошадей и их активностей: кормление, прогулки в левадах, тренировки и др.

- Минимум экранов: список лошадей, карточка лошади с лентой активностей, форма добавления активности
- Локальная БД: Room
- UI: Jetpack Compose Material 3
- Навигация: Navigation Compose

## Возможности
- **Лошади**: добавление, просмотр
- **Активности**: добавление (кормление, левада, тренировка, ветеринар, другое), заметки, длительность, время
- **Лента** активностей для конкретной лошади

## Быстрый старт
1) Откройте проект в Android Studio (Giraffe/Koala+)
2) Убедитесь, что установлены:
   - Android Gradle Plugin 8.5+
   - JDK 17
   - Kotlin 2.0+
   - Android SDK 34
3) Запустите сборку/запуск: Run app (или `./gradlew assembleDebug`)

Примечание: В удалённой среде сборка может не выполниться из‑за отсутствия Android SDK. Рекомендуем собирать локально в Android Studio.

## Структура
- `app/src/main/java/com/example/stabletracker`
  - `data` — Room сущности, DAO, база, репозиторий
  - `ui` — экраны и тема
  - `viewmodel` — ViewModel для экранов
- `app/src/main/res/values` — базовые ресурсы

## Минимальный функционал
- Список лошадей с кнопкой «Добавить»
- Карточка лошади с лентой активностей
- Форма добавления активности (тип, время, длительность, заметки)
